# ゴミ箱ゲーム MVP仕様

## 概要
プレイヤーがゴミを投げ、遠くのゴミ箱に入れる3Dゲーム。

## MVPで作るもの
- 横画面
- 3D空間
- ゴミを投げる
- ゴミ箱に入ったら成功
- 距離に応じてスコア
- 投げた後、カメラは固定レール方式で追従する

## カメラ方針
座標を x, y, z とする。

- カメラのxは固定
- カメラのyは固定
- カメラのzだけゴミのz位置に同期する
- カメラは常にゴミを画面中央付近に映す
- 完全追従カメラではなく、レール上を移動する固定カメラにする
- 酔い防止を優先する

## 最初の操作
- マウスドラッグまたはスワイプで投げ方向と強さを決める
- 離したら投げる
- Rキーまたはボタンでリトライ

## 最初のステージ
- 手前に投げる位置
- 奥にゴミ箱
- 途中に低い障害物
- 床はシンプル
- まずは1ステージのみ

## 優先順位
1. 投げる気持ちよさ
2. ゴミ箱に入る判定
3. カメラの見やすさ
4. ステージレイアウト
5. 見た目

## 最初は使わないもの
- 課金
- ランキング
- オンライン対戦
- 高品質アセット
- 複雑なステージギミック
