# 缶シュート MVP仕様

## 概要
飲み終わった空き缶を、街中にある缶用ゴミ箱の丸穴へ投げ込む3D物理ゲーム。

## コンセプト
狙って、投げて、スパッと穴に入れる。
ピンボールのような複雑なギミックは入れない。
気持ちよさは「物理挙動」「穴に入る瞬間」「成功演出」で作る。

## 舞台
背景は屋外。
最初は仮の街中ステージでよい。
自販機の横にある缶用ゴミ箱をイメージする。

## 投げるもの
- 飲み終わった空き缶
- MVPではCylinderでよい
- 後でBlenderで低ポリ缶モデルを作る

## ゴール
- 缶用ゴミ箱
- 上部または正面上部に丸い投入口がある
- 缶が穴に入ったら成功
- MVPでは簡易モデルでよい
- 投入口は黒い円で分かりやすくする

## 操作
- マウス左ボタンを押す
- ドラッグして方向と強さを決める
- 離したら缶を投げる
- Rキーでリトライ

## ステージ構成
3ステージ構成。

### Stage 1
シンプルで簡単。
投入口が大きく、距離も近い。

### Stage 2
少し難しい。
距離が伸びる、または投入口が少し狭い。

### Stage 3
めっちゃ難しい最終ステージ。
距離が長い、角度が難しい、後で風の影響を入れる候補。

## 残機
- 初期残機は8個
- 1投ミスするたびに残機が1減る
- 残機が0になるとゲームオーバー
- 残機数は後で調整する可能性あり

## クリアランク
クリア時の残機数でランクを決める。

仮ランク:
- 残機8: S
- 残機6〜7: A
- 残機4〜5: B
- 残機1〜3: C

## 後で追加する要素
- 風
- 風向き表示
- 缶の回転
- 成功時の気持ちいいSE
- ステージごとの背景
- Blender製の缶、自販機、缶用ゴミ箱モデル

## MVPでまだ入れないもの
- ピンボール要素
- 複雑な障害物
- オンラインランキング
- 課金
- 高品質モデル
- 複雑なUI
