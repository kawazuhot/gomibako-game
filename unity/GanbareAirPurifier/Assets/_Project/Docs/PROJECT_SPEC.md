# 頑張れ！空気清浄！ PROJECT_SPEC

## ゲーム概要

画面中央下の空気清浄機で、画面中央上に出現するアイテムを吸引するカジュアルゲーム。

クリア式ではなく、制限時間内にスコアとコンボを伸ばすハイスコア型ゲームとする。

吸引Lvに応じて吸えるアイテムが増える。吸えるアイテムを吸うと吸引ゲージが増え、ゲージMAXで吸引Lvが上がる。吸えないアイテムを吸うと失敗し、秒数ペナルティとコンボリセットが発生する。

## 絶対デザインルール

- 登場するキャラクター・アイテムはすべて白縁あり。
- かわいいポップなデザイン。
- 細かい描き込みはしない。
- 視認性を最優先する。
- 小さいスマホ画面でも一瞬で判別できること。

## 初期実装範囲

まずは以下の2ステージまで実装する。

1. 家ステージ
2. 街ステージ

将来的に以下を追加できる構造にする。

3. 都市ステージ
4. 宇宙ステージ

## ステージと吸引Lv

### 家ステージ

- Lv1 = 小
- Lv2 = 中
- Lv3 = 大

### 街ステージ

- Lv4 = 小
- Lv5 = 中
- Lv6 = 大

吸引Lvが4になったら、家ステージから街ステージへ切り替える。

## アイテム例

### 家ステージ

- Lv1: ホコリ、紙くず
- Lv2: ティッシュ、ペン、缶
- Lv3: クッション、椅子、ぬいぐるみ

### 街ステージ

- Lv4: 看板、小さい植木、空き缶
- Lv5: 自転車、信号、ベンチ
- Lv6: 車、小さい家

## サイズルール

アイテムの吸いやすさは、画像サイズで表現する。

- 小 = scale 0.75
- 中 = scale 1.0
- 大 = scale 1.75
- 現在の吸引Lvより高いアイテムは、少し大きく見せる。

## 吸引成功

- アイテムが一度上にふわっとポップする。
- その後、空気清浄機に吸い込まれる。
- 吸引ゲージが増える。
- スコアが増える。
- コンボが+1される。
- ゲージMAXで吸引Lvが上がる。

## 吸引失敗

- アイテムが一度上にふわっとポップする。
- 吸えないアイテムは空気清浄機の前まで来る。
- 空気清浄機が震える。
- アイテムが横に弾き飛ばされる。
- 秒数ペナルティが入る。
- コンボが0になる。

## 空気清浄機の演出

- 吸引中は少し縦長に伸ばす。
- 軽く拡大縮小する。
- 成功時は気持ちよく吸い込む。
- 失敗時は横にブルブル震える。

## 早送り

- 早送りボタンは必須。
- 押している間、ゲーム進行が速くなる。
- 吸引中でも早送り可能。
- ただし、早送り中はミスしやすくなる。

## UI

以下を表示する。

- TIME
- SCORE
- COMBO
- 吸引Lv
- 吸引ゲージ
- 現在ステージ名
- 早送りボタン

特に吸引Lvは、画面上部の見やすい位置に大きく表示する。

## 制限時間

- 仮で90秒。

## 実装方針

- 物理演算ではなく、DOTweenによる座標アニメーションで実装する。
- まずは仮素材でよい。
- 仮素材は白縁付き、名前入りでわかりやすくする。
- 見た目より、ゲームループが動くことを優先する。

## 実装・確認・修正ループ

実装後は、単にコードを書いて終了せず、Unity上で挙動確認し、問題があれば修正して再確認する。

### 確認するポイント

- ゲーム開始時に家ステージが表示されるか。
- 画面中央下に空気清浄機が配置されているか。
- 画面中央上にアイテムが出現するか。
- アイテムが一度上にふわっとポップするか。
- 吸引時にアイテムが空気清浄機へ向かって吸い込まれるか。
- 吸引中に空気清浄機が少し縦長に伸び、軽く拡大縮小するか。
- 現在の吸引Lv以上のアイテムは成功扱いになるか。
- 現在の吸引Lvより高いアイテムは失敗扱いになるか。
- 成功時に吸引ゲージ、スコア、コンボが増えるか。
- ゲージMAXで吸引Lvが上がるか。
- 吸引Lv4で家ステージから街ステージに切り替わるか。
- 失敗時に空気清浄機が横に震えるか。
- 失敗時にアイテムが横に弾き飛ばされるか。
- 失敗時に秒数ペナルティが入るか。
- 失敗時にコンボが0になるか。
- 早送りボタンを押している間、ゲーム進行が速くなるか。
- 早送り中でも吸引できるか。
- TIME、SCORE、COMBO、吸引Lv、吸引ゲージ、現在ステージ名、早送りボタンが表示されているか。
- 特に吸引Lvが大きく、見やすい位置に表示されているか。

### 挙動が微妙な場合に調整する項目

- アイテムのポップする高さ。
- アイテムの吸い込まれる速度。
- 吸引成功時のテンポ。
- 失敗時に弾き飛ばされる方向と速度。
- 空気清浄機の伸縮量。
- 空気清浄機の震え方。
- ゲージ増加量。
- レベルアップまでのテンポ。
- 早送り速度。
- UIの大きさと配置。

### 完了条件

- 家ステージから街ステージまで、ゲームループが一通り動く。
- 吸引成功と吸引失敗の違いが見た目でわかる。
- 吸引Lvによる吸える／吸えない判定が機能している。
- アイテムサイズによって吸いやすさが直感的に伝わる。
- コンボ、スコア、タイマー、ゲージ、ステージ切り替えが機能している。
- Unity Consoleに重大なエラーが出ていない。
- プレイしていて明らかに破綻している挙動がない。
